# Portfolio

My personal portfolio site — hand-built with HTML and CSS, hosted free on GitHub Pages.

## Live site

Once GitHub Pages is enabled, this is published at:
https://concept94.github.io/portfolio/

## Editing it

- `index.html` — the page content (replace **Your Name** and tweak the text)
- `style.css` — the colours, fonts, and layout

Edit the files, commit, and push — GitHub Pages updates the live site automatically within a minute or two.

## Local preview

Just open `index.html` in any web browser to see your changes before pushing.
